# Retail Sales Data Analysis

## Project Overview
This project involves a comprehensive analysis of a retail sales dataset. The objective was to clean the raw data, perform exploratory data analysis, and visualize key performance metrics to gain actionable business insights.

## Tools Used
- **Microsoft Excel**: Data cleaning, formula-based analysis (SUMIF, MIN, MAX, AVERAGE), and visualization.
- **GitHub**: Version control and project documentation.

## Project Structure
The Excel file is organized into the following sheets:
1. **Part 01_Understand_Dataset**: Basic dataset overview (row/column counts, data types).
2. **Part 03_Data Quality Check**: Validation checks for blanks, duplicates, and formula accuracy.
3. **Part 02_Cleaned Data**: The final cleaned dataset used for all calculations.
4. **Part 04_Basic Analysis**: Summary statistics and category-based sales calculations.
5. **Charts**: Visual representation of data including bar and pie charts.
6. **Part 05_Sorting and filter ss**: Screenshots demonstrating filtering and sorting operations.
7. **Observations**: A summary of key findings derived from the analysis.

## Key Insights
Based on the analysis, the following observations were made:
- **Highest Sales Category**: Electronics generated the highest total sales (156,905).
- **Lowest Sales Category**: Beauty recorded the lowest total sales (143,515).
- **Sales by Gender**: Female customers contributed higher total sales (232,840) compared to male customers (223,160).
- **Customer Demographics**: The customer age ranges from 18 to 64 years.
- **Total Performance**: A total of 1,000 transactions were processed, with 2,514 units sold.
